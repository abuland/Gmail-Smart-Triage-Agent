# Gmail Smart Triage AI Agent

An AI-powered Gmail automation tool that reads, classifies, labels, and organizes incoming emails using Python, the Gmail API, and OpenAI.

## Features

- Secure OAuth 2.0 Gmail authentication
- AI-powered email classification
- Automatic Gmail label creation
- Inbox organization & auto-mark-as-read
- Modular architecture

## Tech Stack

- Python
- Gmail API
- OpenAI API
- OAuth 2.0
- dotenv

## Setup

1. Clone repo
2. Create virtual environment
3. Install dependencies:
   pip install -r requirements.txt
4. Add `.env` file with:
   OPENAI_API_KEY=your_key_here
5. Add Google `credentials.json`
6. Run:
   python main.py
